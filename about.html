About

Recipe Roulette solves “choice overload” by giving you one cookable recipe per spin, with optional filters (ingredient, category, area).

Stack: HTML/CSS + vanilla JS (frontend), Vercel Serverless Functions (backend), TheMealDB (dev key=1), Supabase (database for favorites).

How it works

[Filters] -> /filter.php -> pick random id -> /lookup.php -> normalize -> render
(No filters) -> /random.php -> normalize -> render
